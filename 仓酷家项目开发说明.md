## 一、产品概述

### 1.1 产品名称与定位

**仓酷家** - 中小仓库的美图秀秀

**核心价值主张**: 仓库户型图 + 3D物流对象资产 = 快速输出3D物流仓库效果图

**产品定位**: 5分钟拖拽输出3D仓库规划效果图和实施清单，不用学CAD，不用等规划部出图，人人都是物流规划师！

### 1.2 目标用户

- 物流仓库规划人员
- 仓储管理人员
- 物流方案销售/售前
- 中小物流企业主

### 1.3 商业价值

| 价值点 | 说明 |
|--------|------|
| **降低门槛** | 无需学习复杂CAD软件，拖拽即可完成规划 |
| **节省时间** | 5分钟输出专业效果图，传统方式需数小时 |
| **即时沟通** | 现场与客户沟通，实时调整方案 |
| **专业输出** | 自动生成实施清单和项目报告 |
| **成本优势** | 减少对外部规划部门的依赖 |

### 3.2 核心功能详解

#### 3.2.1 2D工作台
- **仓库绘制**: 支持多边形自由绘制和矩形快捷生成
- **功能区规划**: 11种类型（收货区、存储区、发货区等）
- **精确编辑**: 8个手柄调整大小，双击输入精确尺寸
- **文字标注**: 支持添加、编辑、删除标注

#### 3.2.2 3D场景渲染
- **固定网格**: 每格1米，100×100米范围，便于直观判断距离
- **坐标轴刻度**: 每1米短刻度，每10米长刻度+数字标签
- **多边形墙体**: 根据2D绘制自动生成3D墙体
- **透明度优化**: 地面网格10%，功能区15%，围墙15%

#### 3.2.3 对齐线功能
- **绘制**: 点击两点生成红色无限长虚线
- **管理**: 左侧面板列表显示，支持删除单条/清空全部
- **吸附**: 10cm阈值，对象边线吸附，无限长直线吸附
- **性能**: 网格空间分区优化（>20条线时5-10倍性能提升）

#### 3.2.4 距离线功能（MVP 1.5新增）
- **定点画线**: 点击起点 → 输入距离 → 鼠标控制方向 → 点击确认
- **实时预览**: 红色预览线跟随鼠标
- **紫色线条**: 固定线使用紫色提高可见性
- **自动管理**: 15秒后自动清除，画完自动切换选择模式

#### 3.2.5 对象操作
- **移动**: 通过【移动】按钮触发，支持对齐线吸附
- **旋转**: 垂直Y轴旋转，支持精确角度输入
- **批量复制**: 行列、间距、方向、旋转、预览
- **对齐工具**: 左对齐、右对齐、居中对齐、水平等距、垂直等距

## 七、技术债务与优化计划

### 7.1 中优先级

**THREE dispose 资源释放**
- **问题**: 模型创建函数缺少 dispose 逻辑
- **影响**: 可能造成内存泄漏
- **计划**: 后续版本添加 disposeModel() 辅助函数

### 7.2 低优先级

**Vector3 对象池（性能优化）**
- **问题**: 大量临时 Vector3 对象创建
- **影响**: 增加 GC 压力
- **计划**: 使用对象池模式复用实例
