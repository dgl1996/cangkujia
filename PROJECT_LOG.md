## 最新开发计划 (2026-03-07)

### 核心功能页重构方案

#### 新工作流程
1. **第一步：创建仓库**
   - 核心功能页默认加载2D画布作为主工作台
   - 在2D画布上手绘仓库形状
   - 设置仓库基本参数（高度、水平相对高度等）

2. **第二步：功能区规划**
   - 在2D画布上添加和编辑功能区
   - 划分收货区、存储区、出库区等功能区域

3. **第三步：生成3D仓库**
   - 点击"生成3D仓库"按钮
   - 设计工作台从2D模式切换到3D模式
   - 基于2D绘制生成3D仓库背景
   - 添加仓库附属对象（墙体、门、窗户等）

4. **第四步：添加3D物流对象**
   - 在3D模式下添加货架、设备等3D物流对象
   - 调整对象位置和参数

5. **第五步：完成项目**
   - 保存项目
   - 生成项目报告
   - 导出3D效果图

#### 关键功能特性
- **工作台模式切换**：2D和3D模式之间的平滑切换
- **双向编辑机制**：在3D模式下可随时切换回2D模式进行编辑，不影响已放置的3D对象
- **实时同步**：2D编辑实时同步到3D场景
- **灵活导出**：在3D模式的任何阶段都可以导出3D效果图

   - 用户可自由切换步骤

   - 鼠标滚轮缩放（0.1x-5x）
   - Shift+拖拽平移画布
   - 左下角坐标标尺（X向右，Y向上）
   - 右下角鼠标坐标显示
   - 右上角缩放控制

#### 修复的问题
- **线段点击检测**：使用画布全局点击+点到线段距离计算，解决SVG事件绑定问题
- **长度输入无效**：修复`confirmEditLine`函数，确保新长度正确赋值给`currentSegmentLength`
- **角度精度**：严格对齐，仅±0.1度容错，避免累积误差
- **视图切换**：步骤切换时自动切换2D/3D视图

#### 技术方案
- 使用`pointToLineDistance`算法检测点击的线段
- 10像素点击容差
- 状态管理：`isEditMode`、`isEditingLine`、`editingLineIndex`
- 画布变换：`canvasScale`、`canvasOffset`实现缩放平移
- CSS transform实现高效渲染
