\### 1.2 顶部项目工具栏（全局操作）



\*\*布局说明\*\*：单行显示，分为左中右三部分



```

┌─────────────────────────────────────────────────────────────────────────────────────────────┐

│ 左侧                                                                          │ 中间                │ 右侧                                        │

├───────────────────────────────────┼───────────┼─────────────────────────────────────────────┤

│ 🏠首页 │ 导入 │ 保存 │ 📷效果图 │ 📄报告 │ ❌关闭 │ 项目名称 │ ◀/▶ │ ▶/◀ │ 🖱️选择 │ 📏测量 │ ➕对齐线 │ ➕📦 │ 📐对齐工具▼ |

└─────────────────────────────────────────────────────────────────────────────────────────────┘

```



\*\*按钮说明\*\*:



| 按钮 | 功能 | 状态 |

|------|------|------|

| 🏠首页 | 返回首页 | 始终可用 |

| 导入 | 导入项目文件 | 始终可用 |

| 保存 | 保存项目 | 始终可用 |

| 📷效果图 | 导出3D效果图 | 生成3D后可用 |

| 📄报告 | 导出项目报告 | 始终可用 |

| ❌关闭 | 关闭项目 | 始终可用 |

| ◀/▶ | 关闭/打开左侧面板 | 始终可用 |

| ▶/◀ | 关闭/打开右侧面板 | 始终可用 |

| 🖱️选择 | 选择模式（默认） | 始终可用 |

| 📏测量 | 测量工具 | 始终可用 |

| ➕对齐线 | 添加对齐线 | 始终可用 |

| ➕📦 | 打开自定义货架（P2页面） | 始终可用 |

| 📐对齐工具▼ | 对齐工具下拉 | \*\*MVP禁用\*\*，与多选功能绑定 |



\*\*特殊说明\*\*：

\- \*\*对齐工具\*\*：当前版本禁用，待多选功能上线后启用。对齐工具包含左对齐、居中、右对齐、水平等距、垂直等距等功能，需要多选功能支持。

\- \*\*➕📦 创建自定义货架\*\*：点击后在新的浏览器标签页打开P2页面（ModelLibrary.vue），用户可以在不离开P1的情况下创建自定义货架。创建完成后，新货架自动同步到P1左侧面板的"我的模型"中。



\*\*删除功能位置\*\*: 右侧面板对象操作面板（按对象类型显示）

\### 1.3 右侧面板（属性 + 对象独有操作）



根据选中对象类型，显示不同内容：



\#### 场景1：未选中任何对象



```

┌─────────────────────┐

│      属性面板        │

├─────────────────────┤

│  仓库全局属性:       │

│  • 长、宽、高        │

│  • 面积             │

│  • 层高             │

│  • 水平高度          │

│  • 功能区数量        │

│  • 放置对象数量      │

└─────────────────────┘

```



\#### 场景2：选中仓库形状（墙体线条）



```

┌─────────────────────┐

│      属性面板        │

├─────────────────────┤

│  仓库形状属性:       │

│  • 线条长度          │

│  • 端点坐标          │

├─────────────────────┤

│    对象操作面板      │

│  \[编辑线条]         │  ← 独有：编辑当前线条

│  \[删除]             │  ← 删除当前线条

└─────────────────────┘

```



\*\*说明\*\*：选中仓库形状的某根线条后，右侧面板显示\[编辑线条]按钮，点击后可编辑该线条的长度。



\#### 场景3：选中功能区



```

┌─────────────────────┐

│      属性面板        │

├─────────────────────┤

│  功能区属性:         │

│  • 名称             │

│  • 类型             │

│  • 尺寸(长x宽)      │

│  • 面积             │

│  • 位置(X,Y)        │

├─────────────────────┤

│    对象操作面板      │

│  \[编辑]             │  ← 进入编辑模式（8手柄+移动）

│  \[删除]             │  ← 删除功能区

└─────────────────────┘

```



\*\*编辑模式说明\*\*:

\- 点击\[编辑]后，功能区显示8个缩放手柄

\- 支持拖拽移动位置

\- 支持输入精确尺寸

\- 按ESC或点击\[完成]退出编辑模式
