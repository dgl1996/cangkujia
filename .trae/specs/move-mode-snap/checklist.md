# Checklist

- [ ] 点击【移动】按钮进入移动模式，对象变为半透明并抬高
- [ ] 移动模式下对象跟随鼠标地面投影点移动（无需按住按键）
- [ ] 对象靠近对齐线10cm内自动吸附到对齐线位置
- [ ] 吸附时对齐线高亮变为绿色
- [ ] 远离对齐线时对象恢复跟随鼠标，对齐线恢复红色
- [ ] 点击鼠标左键确认放置，退出移动模式
- [ ] 按ESC键取消移动，对象回到原位
- [ ] 移动模式下左键不触发画布旋转（OrbitControls左键禁用）
- [ ] 移动模式下显示提示信息"移动模式：点击放置，ESC取消"
- [ ] 退出移动模式后对象恢复不透明状态
