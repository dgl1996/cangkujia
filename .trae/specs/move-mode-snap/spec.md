# 移动模式吸附功能集成 Spec

## Why
当前吸附功能与3D画布旋转存在鼠标左键冲突：用户选中对象后按住左键移动对象时，很容易误触发场景旋转，导致难以精确放置对象。需要将吸附功能集成到现有的【移动】按钮流程中，消除操作冲突。

## What Changes
- 修改移动模式逻辑：从"按住拖拽"改为"鼠标跟随+点击确认"
- 在移动模式下集成对齐线吸附功能
- 添加对齐线高亮反馈（靠近时变绿）
- 添加对象视觉反馈（半透明/抬高）
- 支持ESC取消移动

## Impact
- Affected specs: 对齐线吸附功能、对象移动操作
- Affected code: ThreeScene.vue（移动模式逻辑）、CoreFunction.vue（移动按钮交互）

## ADDED Requirements

### Requirement: 移动模式鼠标跟随
The system SHALL 在进入移动模式后，让选中对象跟随鼠标地面投影点移动，无需按住鼠标按键。

#### Scenario: 进入移动模式
- **GIVEN** 用户已选中一个3D对象
- **WHEN** 用户点击右侧面板的【移动】按钮
- **THEN** 系统进入移动模式，对象变为半透明并稍微抬高
- **AND** 对象开始跟随鼠标在地面上的投影点移动

#### Scenario: 鼠标跟随移动
- **GIVEN** 系统处于移动模式
- **WHEN** 用户移动鼠标（不按任何键）
- **THEN** 对象实时跟随鼠标地面投影点位置
- **AND** 对象Y坐标保持为0（贴地）

### Requirement: 移动模式吸附功能
The system SHALL 在移动模式下，当对象靠近对齐线时自动吸附到对齐线位置。

#### Scenario: 靠近对齐线触发吸附
- **GIVEN** 系统处于移动模式
- **WHEN** 对象中心点距离对齐线小于10cm
- **THEN** 对象自动"跳"到对齐线位置（最近点）
- **AND** 对齐线高亮变为绿色
- **AND** 对象保持半透明状态

#### Scenario: 远离对齐线取消吸附
- **GIVEN** 系统处于移动模式且对象已吸附到对齐线
- **WHEN** 对象中心点距离对齐线大于10cm
- **THEN** 对象恢复跟随鼠标位置
- **AND** 对齐线恢复原始颜色（红色）

### Requirement: 移动模式确认与取消
The system SHALL 支持点击确认放置或ESC取消移动。

#### Scenario: 点击确认放置
- **GIVEN** 系统处于移动模式
- **WHEN** 用户点击鼠标左键
- **THEN** 对象固定在最终位置
- **AND** 退出移动模式
- **AND** 对象恢复不透明状态
- **AND** 对齐线恢复原始颜色

#### Scenario: ESC取消移动
- **GIVEN** 系统处于移动模式
- **WHEN** 用户按下ESC键
- **THEN** 对象回到移动前的原始位置
- **AND** 退出移动模式
- **AND** 对象恢复不透明状态
- **AND** 对齐线恢复原始颜色

## MODIFIED Requirements

### Requirement: 现有移动模式
**原逻辑**: 选中对象后按住鼠标左键拖拽移动
**新逻辑**: 点击【移动】按钮进入移动模式，鼠标跟随移动，点击确认或ESC取消

**理由**: 消除鼠标左键与画布旋转的冲突

## REMOVED Requirements
无
