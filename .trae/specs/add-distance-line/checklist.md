# Checklist

## Phase 1: UI组件开发

### 工具栏按钮检查项

- [ ] "+距离线"按钮在工具栏正确显示
- [ ] 按钮有合适的图标和文字
- [ ] 按钮样式与其他工具按钮一致
- [ ] 点击按钮进入距离线模式
- [ ] 按钮在距离线模式下显示激活状态
- [ ] 再次点击按钮退出距离线模式

### 距离输入弹窗检查项

- [ ] 弹窗在起点选择后正确显示
- [ ] 弹窗包含距离输入框
- [ ] 弹窗包含确认按钮
- [ ] 弹窗包含取消按钮
- [ ] 输入框默认值为空或有提示
- [ ] 输入框接受小数（如1.5）
- [ ] 输入范围验证：小于0.1米时提示错误
- [ ] 输入范围验证：大于100米时提示错误
- [ ] 点击确认关闭弹窗并进入预览模式
- [ ] 点击取消关闭弹窗并退出距离线模式

## Phase 2: ThreeScene 核心逻辑开发

### 距离线模式状态管理检查项

- [ ] `isDistanceLineMode` 状态变量正确工作
- [ ] `distanceLineStartPoint` 正确存储起点
- [ ] `distanceLinePreview` 正确存储预览线
- [ ] `distanceValue` 正确存储输入距离
- [ ] `startDistanceLineMode` 函数正确进入模式
- [ ] `stopDistanceLineMode` 函数正确退出模式
- [ ] ESC键正确取消距离线模式
- [ ] 距离线模式与其他模式互斥（测量、对齐线）

### 起点选择和距离输入检查项

- [ ] 距离线模式下点击地面正确选择起点
- [ ] 起点坐标Y值为0（地面水平面）
- [ ] 起点选择后正确触发弹窗显示
- [ ] 距离值正确传递给预览逻辑

### 距离线预览检查项

- [ ] 预览线从起点开始显示
- [ ] 预览线长度固定为输入距离
- [ ] 预览线方向随鼠标移动而变化
- [ ] 预览线显示在地面之上（Y>0）
- [ ] 预览线使用虚线或不同颜色（与固定线区分）
- [ ] 预览线实时更新，无卡顿

### 终点确定和距离线创建检查项

- [ ] 预览模式下点击正确确定终点
- [ ] 固定距离线使用实线显示
- [ ] 距离标签正确显示距离值（如"1.5米"）
- [ ] 距离标签位置在距离线上方
- [ ] 距离标签文字清晰可读
- [ ] 10秒倒计时正确启动
- [ ] 退出距离线模式

### 距离线管理和清理检查项

- [ ] `distanceLines` 数组正确存储多个距离线
- [ ] 多个距离线可以同时显示
- [ ] 每个距离线独立计时
- [ ] 10秒后距离线自动清除
- [ ] 距离线清除时释放资源（geometry, material）
- [ ] 无内存泄漏

- [ ] 距离线模式与测量模式互斥
- [ ] 距离线模式与对齐线模式互斥
- [ ] 距离线模式与移动模式互斥

- [ ] 多个距离线同时存在测试
- [ ] ESC取消测试
- [ ] 输入验证测试（边界值0.1米、100米）
- [ ] 测量工具功能不受影响
- [ ] 对齐线功能不受影响
- [ ] 移动对象功能不受影响
