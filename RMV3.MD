## 项目基本信息

### 项目名称
**仓酷家** - 中小仓库的美图秀秀

### 核心价值主张
仓库户型图 + 3D物流对象资产 = 快速输出3D物流仓库效果图

### 产品定位
5分钟拖拽输出3D仓库规划效果图和实施清单，不用学CAD，不用等规划部出图，人人都是物流规划师！

### 创始人
**安德森（董国礼）** - 26年物流行业经验

---

## 技术架构

### 前端 (Frontend)
- **框架**: Vue 3 + Vite
- **3D引擎**: Three.js
- **UI组件**: 自定义组件 + Element Plus
- **端口**: 5173

### 后端 (Backend)
- **框架**: FastAPI
- **语言**: Python
- **端口**: 8000

### 3D模型系统
- **格式**: GLB/GLTF
- **数量**: 40个模型文件
- **位置**: `/frontend/public/assets/models/`
- **生成工具**: Python + Trimesh
- **自动添加工具**: `scripts/model_generator_template.py` (正式命名: **3D对象自动添加工具**)

### 坐标系与单位
- **Three.js**: Y轴向上，单位cm
- **Trimesh**: Z轴向上，单位mm
- **转换**: -90度绕X轴旋转，缩放0.1

## 核心功能模块

### 1. 2D工作台功能 ✅
- 2D仓库形状绘制（精确绘制、编辑）
- 快捷生成矩形仓库
- 仓库参数设置（高度、水平高度）
- 11种功能区类型
- 拖拽创建矩形功能区
- 8个手柄调整大小
- 双击输入精确尺寸
- 面积实时显示
- 文字标注功能

### 2. 3D场景基础功能 ✅
- 场景渲染（背景色#f0f0e6）
- 相机控制（OrbitControls）
- 窗口自适应
- 多边形墙体生成
- 功能区3D可视化（虚线边框、名称标签）
- XZ坐标轴指示器
- 地面网格

### 3. 3D物流对象库 ✅
**9大分类，40个对象:**

1. **仓库附属设施** (5个): F001-F005 (门、窗、半透明玻璃幕墙墙体、工业滑升门、工业采光窗)
2. **货架系统** (12个): A101-A112
3. **载具容器** (8个): C101-C108
4. **搬运设备** (6个): B101-B106
5. **输送设备** (3个): D101-D103
6. **拣选设备** (3个): E101-E103
7. **分拣设备** (2个): H101-H102
8. **其他设备** (2个): I101-I102
9. **人员** (1个): G101

### 4. 对象操作功能 ✅
- 选择模型（单击选中）
- 移动模型
- 旋转模型（垂直Y轴旋转）
- 删除模型
- 复制模型
- 批量复制（行列、间距、旋转、预览）
- 对齐工具（左对齐、右对齐、居中对齐、水平等距、垂直等距）

### 5. 项目管理功能 ✅
- 保存仓库形状
- 保存功能区规划
- 保存3D物流对象（位置、旋转、缩放）
- 导入项目并恢复所有数据
- File System Access API支持覆盖保存

- 生成3D仓库

### 卡片2: 放置3D对象 → 3D画布
- 9大分类对象库
- 拖拽添加到3D场景
- 对象操作（移动、旋转、删除、复制）
- 批量复制（行数、列数、行间距、列间距、旋转角度、预览、生成）
- 对齐工具（左对齐、右对齐、居中对齐、水平等距、垂直等距）

2. **右侧面板**：专注于"查看和编辑"操作
3. **顶部栏**：项目级别的全局操作
4. **交互逻辑**：选中对象 → 右侧显示属性和操作；未选中 → 显示仓库全局属性
